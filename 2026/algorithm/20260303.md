# 2️⃣ 핵심 개념 정리 (What)

## 🔹 스택 (Stack)

### 정의

스택은 **LIFO (Last In First Out)** 구조를 가지는 선형 자료구조이다.  
가장 나중에 들어온 데이터가 가장 먼저 나간다.

### 기본 연산

- push : 데이터 삽입
- pop : 데이터 제거
- peek : 최상단 데이터 조회

### 핵심 포인트

- Top 포인터를 통해 접근
- 삽입/삭제 모두 O(1)
- 제한된 접근 구조 (Top만 접근 가능)

### 왜 중요한가?

- 함수 호출 스택 (Call Stack)
- 괄호 검사
- Undo/Redo 기능
- DFS 구현

------------------------------------------------------------------------

## 🔹 큐 (Queue)

### 정의

큐는 **FIFO (First In First Out)** 구조를 가지는 선형 자료구조이다.  
가장 먼저 들어온 데이터가 가장 먼저 나간다.

### 기본 연산

- enqueue : 데이터 삽입
- dequeue : 데이터 제거
- peek : 가장 앞 데이터 조회

### 핵심 포인트

- Front / Rear 포인터 사용
- 삽입/삭제 O(1)
- 대기열 처리에 적합

### 왜 중요한가?

- 메시지 브로커
- 작업 스케줄링
- BFS 구현
- 이벤트 처리 시스템

------------------------------------------------------------------------

# 3️⃣ 내부 동작 원리 (How)

## 🔹 스택 동작 과정

### 1️⃣ Push

1. top 증가
2. stack[top] = data

### 2️⃣ Pop

1. data = stack[top]
2. top 감소

### 3️⃣ 예외 상황

- Overflow: 스택 가득 참
- Underflow: 스택 비어 있음

------------------------------------------------------------------------

## 🔹 큐 동작 과정

### 1️⃣ Enqueue

1. rear 증가
2. queue[rear] = data

### 2️⃣ Dequeue

1. data = queue[front]
2. front 증가

### 3️⃣ 선형 큐의 문제

- front가 계속 증가하여 메모리 낭비 발생

### 해결 방법 → 원형 큐 (Circular Queue)

- (rear + 1) % size
- (front + 1) % size
- 메모리 재사용 가능

# 5️⃣ 코드 구현

## 🔹 배열 기반 스택 구현 (Java)

```java
class ArrayStack {
    private int[] stack;
    private int top = -1;
    private int capacity;

    public ArrayStack(int capacity) {
        this.capacity = capacity;
        stack = new int[capacity];
    }

    public void push(int value) {
        if (top == capacity - 1) {
            throw new RuntimeException("Stack Overflow");
        }
        stack[++top] = value;
    }

    public int pop() {
        if (top == -1) {
            throw new RuntimeException("Stack Underflow");
        }
        return stack[top--];
    }

    public int peek() {
        if (top == -1) {
            throw new RuntimeException("Stack is Empty");
        }
        return stack[top];
    }

    public boolean isEmpty() {
        return top == -1;
    }
}
```
## 🔹 연결리스트 기반 큐 구현 (Java)

```java
class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
    }
}

class LinkedQueue {
    private Node front;
    private Node rear;

    public void enqueue(int value) {
        Node newNode = new Node(value);

        if (rear == null) {
            front = rear = newNode;
            return;
        }

        rear.next = newNode;
        rear = newNode;
    }

    public int dequeue() {
        if (front == null) {
            throw new RuntimeException("Queue Underflow");
        }

        int value = front.data;
        front = front.next;

        if (front == null) {
            rear = null;
        }

        return value;
    }

    public int peek() {
        if (front == null) {
            throw new RuntimeException("Queue is Empty");
        }
        return front.data;
    }

    public boolean isEmpty() {
        return front == null;
    }
}
```
